●操作方法
  【キーボード】
  W/A/D/S       : 移動
  Q/E           : カメラ左右操作
  SPACE         : ジャンプ
  J             : ブロック設置
  F + J         : 1マス下にブロック設置
  L             : ブロック破壊

【ゲームパット(DirectInput LogicoolF310)】
 左スティック   : 移動
 右スティック   : カメラ左右操作
 A              : ジャンプ
 X              : ブロック設置
 RB + X         : 1マス下にブロック設置
 B              : ブロック破壊
